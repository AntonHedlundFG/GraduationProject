12/* This struct can be expanded to include more settings.
13* Make sure the settings are implemented just like NumberOfPlayers
14* 1. Declare the UPROPERTY().
15* 2. #define a macro for its name, it will make sure there are no spelling errors in FStrings.
16* 3. Update GetOptionsStringAppendage() to AddOption() the new property. It's good practice to not include un-assigned properties.
17* After these steps are done, when you load a level with these settings, you can fetch the stored properties
18* in the GameMode by using UGameplayStatics::GetIntOption(OptionsString, YOUR_MACRO, 0), and similar functions.
19*/
