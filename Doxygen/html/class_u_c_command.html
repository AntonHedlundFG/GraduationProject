Detailed Description

Commands should not be created and executed manually. Instead, you should override the GenerateAbilityCommand() of a UCItem to create the command, which will then be executed and undone automatically by the GameMode. Note that things that trigger as a -result- of using a command, rather than a deliberate action from a Controller, should be created as a UCConsequence and registered in the CGameMode with RegisterAndExecuteConsequence()
