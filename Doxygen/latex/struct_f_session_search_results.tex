\doxysubsection{Detailed Description}
This is a list of all search results from the "{}\+Find Games"{} button. It contains everything that should be presented in the UI to determine Which game to join. The index of the selected game in this list is used to call the Join\+Session(int index) function. For UI\+: subscribe to On\+Session\+Search\+Update to know when to redraw UI, find the actual list in Session\+Search\+Results.
