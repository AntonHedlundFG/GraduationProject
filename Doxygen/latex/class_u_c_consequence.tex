\doxysubsection{Detailed Description}
\doxylink{class_u_c_consequence}{UCConsequence} is an event that trigger as a -\/result-\/ of using a command, rather than a deliberate action from a Controller. It should be created and registered in the CGame\+Mode with Register\+And\+Execute\+Consequence(). Do not execute it manually.
